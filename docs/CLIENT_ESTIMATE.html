Пропозиція до реалізації

LinkedIn Marketplace
& Rental — MVP

Онлайн-сервіс для продажу SMM-послуг та оренди готових LinkedIn-акаунтів. Готовий до запуску за 4 тижні.

Термін ~4 тижні
Робочих днів ~19
Локалізація EN + UA

Що ми робимо

Веб-сайт + адмін-панель, де клієнт може:

  • Купити SMM-послугу — наприклад, пакет connections (25 / 50 / 75 / 100) для своєї LinkedIn-сторінки
  • Орендувати готовий LinkedIn-акаунт на 1 місяць — з логіном, паролем, проксі і 2FA
  • Бачити свої замовлення в особистому кабінеті, отримувати 2FA-коди, продовжувати оренду

Адмін керує всім з однієї панелі: акаунти, замовлення, статуси, сповіщення в Telegram з кнопками "Прийняти / Відхилити".

Що отримає клієнт по тижнях

1 Сайт + основа
Лендінг (EN + UA) Головна з двома кнопками "Buy SMM" / "Rent Accounts", блок "Як це працює", FAQ, контакти. Адмінка для редагування.
Реєстрація і логін Email + підтвердження + скидання пароля.
Шифрування даних Паролі, проксі, 2FA-секрети зашифровані в базі (AES-256).
Інфраструктура Налаштовано базу даних, домен, деплой, окремий сервер для авто-задач.
2 Маркетплейс
Каталог SMM-послуг Старт — Connections. Пакети 25 / 50 / 75 / 100. Якщо немає вільних акаунтів — великий пакет неактивний.
Каталог акаунтів для оренди Картки з гео, кількістю connections, ціною. Фільтри. Поштучний вибір або готові пакети.
Форма замовлення Заявка → email + Telegram адміну з кнопками "Прийняти / Відхилити".
База акаунтів в адмінці CRUD + CSV-імпорт. Поля: login, password, proxy, 2FA, гео, статус, пул (SMM / Rental).
3 Кабінет + сповіщення
Особистий кабінет "Мої замовлення" + "Мої акаунти" з кнопками: показати дані, показати 2FA-код, продовжити, поскаржитися.
Адмін-дашборд Кількість акаунтів по пулах і статусах. Bulk-дії, ручна заміна, перегляд "хто і до коли орендує".
Email-сповіщення 8 типів: підтвердження, видача, нагадування 3д / 1д, експірація, авто-заміна.
Telegram-бот Алерти адміну з inline-кнопками "Прийняти / Відхилити".
4 Автоматика + запуск
Авто-зміна пароля При експірації — сервіс заходить на LinkedIn, змінює пароль, оновлює 2FA. Старі дані юзера невалідні.
ADSPower / LinkedinHelper sync Раз на годину пулить стан акаунтів, переключає статус "warming → ready" автоматично.
Авто-заміна забанених Якщо акаунт забанили під час оренди — автоматично видається інший з пулу і гео.
Тестування + Deploy Перевірка сценаріїв, мобільна версія, фікс багів, production-deploy + інструкція для адміна.

Що може зсунути термін

Фактор Вплив
LinkedIn змінив UI — Playwright-скрипти треба переписувати +0.5-1 тиждень
Клієнт хоче кастомний дизайн з кількома раундами правок +1 тиждень
Нестабільні проксі / зміна провайдера +0.5 тижні
ADSPower вже працює і налаштований у клієнта −0.5 тижні

Що буде гарантовано на запуск

  • Працюючий лендінг (EN + UA)
  • Реєстрація юзерів + кабінет
  • Каталог SMM-послуг з capacity-перевіркою
  • Каталог акаунтів з фільтрами
  • Email + Telegram сповіщення
  • Inline-кнопки "Прийняти / Відхилити" в Telegram
  • Reveal credentials у кабінеті
  • Кнопка "Show 2FA code" (генерація на льоту)
  • Auto-нагадування 3д / 1д
  • Auto-зміна пароля при експірації
  • Auto-заміна забаненого акаунта
  • ADSPower sync статусів
  • Production deploy + інструкція для адміна
